# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is **ClaudeCodeHandbook** — a comprehensive Turkish-language educational handbook covering AI fundamentals, Large Language Models, and Claude Code usage. It is a pure documentation project (156 Markdown files, no application code to build or test).

**Author:** İzzet Yasin ATEŞ | **License:** MIT

## Content Structure

21 numbered chapters organized in progressive complexity:

- **01–05 (Temel Bilgiler):** AI fundamentals, LLMs, providers, AI-driven development, Claude ecosystem
- **06–10 (Claude Code):** Introduction, UI/commands, 30+ tools, memory/context, permissions/security
- **11–14 (Genişletme):** MCP, Skills/Plugins, Subagents, Hooks/Automation
- **15–18 (Entegrasyon & Operasyon):** IDE integrations, CI/CD, configuration, enterprise usage
- **19–21 (Pratik):** 12 role-based guides, practical scenarios, troubleshooting

Each chapter has its own `README.md` with a Mermaid flowchart, content table with reading times, prerequisites, and navigation links.

## Language and Writing Conventions

- All content is written in **Turkish**. New or edited content must also be in Turkish.
- Headings follow a consistent hierarchy: H1 = chapter title, H2 = section, H3 = subsection.
- Mermaid diagrams are used for visual navigation and flowcharts in chapter READMEs.
- Reading time estimates are provided per topic file.
- Cross-references use relative Markdown links between chapters.
